# Phrasendrescher - Fußballerphrasen-Generator

Ein einfacher, aber genialer "Phrasendrescher" für alle Fußballfans. Drück den Knopf und lass dich von tiefgründigen Weisheiten wie "Mailand oder Madrid - Hauptsache Italien!" inspirieren.

## Features

- **Über 240 Phrasen**: Von Klassikern bis zu modernen Stilblüten.
- **Retro-Design**: Stadion-Scoreboard Look mit LED-Ticker.
- **Responsive**: Funktioniert auf Desktop und Mobile.
- **Dark Mode**: Automatische Anpassung an deine Systemeinstellungen.
- **Kopierfunktion**: Teile die besten Sprüche direkt mit Freunden.

## Nutzung

Einfach die `index.html` im Browser öffnen und los geht's!

## Technologien

- HTML5
- CSS3 (Custom Properties, Flexbox, Animations)
- Vanilla JavaScript
