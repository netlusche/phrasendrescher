# Phrasendrescher 1.2 - Die Live-Übertragung

Ein interaktiver "Phrasendrescher" für alle Fußballfans, der das Stadion direkt auf deinen Bildschirm bringt. Drücke den Knopf und lass dich von tiefgründigen Weisheiten wie "Mailand oder Madrid - Hauptsache Italien!" im TV-Broadcast-Look inspirieren.

## Features (Neu in v1.2.0)

- **Die Matchday-Engine:** Generiert dynamische Spielereignisse, Nachspielzeiten und Live-Kontexte zu den Zitaten.
- **Broadcast-Design:** Hochwertiges Scoreboard, Live-Ticker und animierte Flutlichter sorgen für echte Stadion-Vibes.
- **Kategorien & Filter:** Sortiere über 140 handverlesene Phrasen nach Trainern, Reportern, Abstiegskampf oder Pressekonferenzen.
- **Metadaten-Tags:** Jede Phrase wird mit Kategorien, Tonalität (Drama, Analyse) und der passenden Ära (90er, Modern) veredelt.
- **Kopierfunktion:** Teile die besten Sprüche direkt mit Freunden.
- **Responsive:** Fluid Layout und optimierte Breakpoints für alle Bildschirmgrößen (Mobile & Desktop).

## Nutzung

Einfach die `index.html` im Browser öffnen und auf "Auf Sendung" klicken!

## Technologien

- HTML5 (Semantisch & Accessible)
- CSS3 (Flexbox, Transforms, Dynamische Keyframe-Animationen)
- Vanilla JavaScript (State-Management, DOM-Manipulation)
